# mech-interp-SAE-demo

A lightweight demo project for mechanistic interpretability using TransformerLens, sparse autoencoders (SAE), and activation patching.

## Project Goal

This repository is a reproducible, learning-oriented demo for studying mechanistic interpretability on small language models.

The project focuses on:
- inspecting internal activations with TransformerLens
- exploring sparse autoencoders as feature decomposition tools
- using activation patching and feature-level interventions for causal analysis
- building a clean GitHub workflow for a reproducible interpretability project

## Scope of V1

The first version of this repository is a demo-oriented reproduction project rather than a novel research project.

V1 will include:
- repository setup and reproducible project structure
- activation inspection on a small language model
- a minimal activation patching example
- SAE loading or training workflow
- feature inspection and a small interpretability case study
